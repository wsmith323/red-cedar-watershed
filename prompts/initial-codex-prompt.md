# Initial Codex Prompt

Read `AGENTS.md` and all documents under `docs/`.

This repository will contain a discussion prototype for a public educational
website about Lake Menomin, Tainter Lake, and the Red Cedar River watershed.

The research materials under `docs/research/` are the evidence base. Treat them
as source material, not as automatically publishable prose.

First:

1. Review the research materials and editorial constraints.
2. Identify the major factual claims that are sufficiently supported for use.
3. Identify contradictions, outdated figures, unsupported claims, and
   unresolved questions.
4. Propose a visitor-oriented site structure.
5. Propose no more than four simple visualizations. For each visualization,
   specify:
   - the question it answers;
   - the source data;
   - the claim it communicates;
   - important caveats.
6. Write `docs/content-plan.md` describing:
   - proposed pages;
   - intended audience and purpose of each page;
   - major claims;
   - required citations;
   - organization credits;
   - proposed visualizations;
   - open editorial decisions;
   - missing information.

Do not implement the website yet.

Do not:

- introduce unsupported facts;
- invent metrics or quotations;
- silently resolve conflicting sources;
- convert modeled estimates into measured facts;
- describe a proposed solution as an adopted plan;
- imply endorsement by any organization;
- make the framing adversarial toward farmers or landowners.

The output of this task should be a content and implementation planning document
that can be reviewed before any site code is written.
