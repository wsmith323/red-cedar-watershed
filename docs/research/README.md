# Research Materials

Place Deep Research outputs and manually collected source material here.

Suggested files:

- `research-report.md`
- `source-inventory.md`
- `organizations.md`
- `open-questions.md`
- `claims-and-evidence.md`

For each important source, capture:

- publishing organization;
- title;
- publication date;
- URL;
- relevant page or section;
- plain-language summary;
- whether the information appears current;
- how it may be used on the site;
- whether it is measured, modeled, reported, interpreted, or unresolved.
